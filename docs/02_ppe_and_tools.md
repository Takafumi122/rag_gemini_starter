# 保護具と工具の取り扱い

工場では多くの保護具（PPE: Personal Protective Equipment）を使用します。保護具は「身を守る最後の砦」です。面倒だからといって省略してはいけません。

### 主な保護具
- 安全帽：頭部を保護する。顎ひもは必ず締める  
- 安全靴：つま先に金属芯が入ったものを使用する  
- 保護手袋：部品の取り扱いや切削時に着用する  
- 保護メガネ：切粉や異物が飛ぶ作業では必須

### 工具の管理
工具は使ったら必ず元の位置に戻します。破損した工具を使うと事故の原因になります。毎日の点検を習慣化し、「異常はすぐ報告」を徹底しましょう。
